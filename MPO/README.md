# Explicación de la arquitectura del proyecto:
· El proyecto Arcadepedia está organizado utilizando una arquitectura por capas, siguiendo una estructura modular basada en el patrón MVC (Model View Controller). Esta organización permite separar las responsabilidades de cada parte de la aplicación, facilitando el mantenimiento, la escabilidad y la comprensión del código.
