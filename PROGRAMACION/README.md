# ¿Qué hace la aplicación?
· La aplicación gestiona un catálogo predefinido de videojuegos, permitiendo consultar las fichas de cada videojuego incluido. También permite el registro y guardado de usuarios.

# ¿Cómo se ejecuta?
· Se ejecuta el archivo principal de la aplicación, que inicia la interfaz gráfica desarrollada con JavaFX. Al arrancar, aparece automáticamente la pantalla de presentación (index.html) desde donde el usuario puede comenzar a navegar por el catálogo.

· También es necesario tener instalado Java y las librerías de JavaFX para que la aplicación funcione correctamente.

# ¿Qué funcionalidades tiene?
· Mostrar una pantalla de bienvenida al iniciar la aplicación.

· Acceder al catálogo principal de videojuegos.

· Navegar entre diferentes páginas de la aplicación.

· Consultar datos adicionales y referencias externas relacionadas con los juegos.

· Ver información sobre los creadores del proyecto en la sección “Sobre nosotros”.

· Volver al menú principal mediante el botón de salida.

· Ofrecer una interfaz inspirada en la estética retro arcade para mejorar la experiencia del usuario.

# ¿Qué entidades gestiona?
· Videojuegos → información detallada de cada juego, como nombre, descripción y datos relacionados.

· Usuarios o visitantes → personas que navegan y consultan el catálogo.

· Creadores/desarrolladores → información relacionada con los autores de los videojuegos o del proyecto.

· Categorías o apartados del catálogo → secciones para organizar y facilitar la navegación del contenido.

# ¿Qué parte usa la base de datos?
· La base de datos se utiliza principalmente en la parte del catálogo de videojuegos.

· Nombre del videojuego.
· Descripción.
· Datos e información detallada.
· Referencias o enlaces relacionados.
· Posibles categorías o clasificaciones.
